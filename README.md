# DSA Problem Tracker

A Java-based console application to manage and track Data Structures & Algorithms problems.

## Features
- Add new DSA problems
- Mark problems as solved
- View progress and problem list
- Simple menu-driven console interface

## Technologies Used
- Java
- Data Structures (ArrayList, HashMap)
- OOP concepts

## How to Run
1. Clone the repository
2. Compile the Java files
3. Run the main class

## Use Case
Helps students track their DSA practice and monitor progress efficiently.
